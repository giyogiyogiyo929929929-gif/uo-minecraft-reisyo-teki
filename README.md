## 美しい数式：オイラーの等式

数学における最も重要な定数たちが、たった一つのシンプルな式で美しく結びついている方程式です。

$$e^{i\pi} + 1 = 0$$

### 構成要素の意味
* **$e$**: ネイピア数（自然対数の底）
* **$i$**: 虚数単位（$i^2 = -1$）
* **$\pi$**: 円周率
* **$1$ と $0$**: 代数学における基本の数

## 無限の神秘：バーゼル問題

自然数の二乗の逆数をすべて足し合わせると、一見関係なさそうな円周率が突然現れる有名な数式です。

$$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{1}{1^2} + \frac{1}{2^2} + \frac{1}{3^2} + \dots = \frac{\pi^2}{6}$$
